# Task 1274 수행 계획서

## 배경

작업지시자는 GitHub issue를 새로 열지 않고 임의 task로 6개 교육 통합 HWP 문서를 PDF 정답지와 전체 비교하라고 지시했다. 비교 중 수식 겹침, 미주 간격 차이, page overflow가 발견되면 페이지 단위로 수정하고, SVG를 PNG로 변환해 Codex가 직접 시각 판단한 뒤 커밋하고 다음 페이지로 진행한다.

이 task는 `task 1261` PR #1264에서 정리한 문28/문8/문12 보정 위에 쌓는다. 해당 보정이 빠지면 `3-10월_교육_통합_2022.hwp`와 `3-09월_교육_통합_2024-미주사이20.hwp`의 이미 확인된 결함이 다시 나타나므로, 현재 기준점은 `local/task_m100_1261` HEAD에서 분기한 `local/task_m100_1274`이다.

## 검증 관점

- 수식과 본문/제목/선택지의 겹침
- 미주 구분선 아래 간격과 미주 사이 간격의 PDF 정답지 대비 차이
- `LAYOUT_OVERFLOW`, `LAYOUT_OVERFLOW_DRAW`, 페이지 하단 잘림
- 좌우 단 흐름에서 문제 제목 또는 꼬리 문단이 한컴 PDF보다 과도하게 내려가거나 올라가는 현상
- 수정 후 다른 대상 문서와 기존 회귀 테스트가 깨지지 않는지 확인

## 진행 방식

1. 각 HWP를 `export-svg`로 페이지별 SVG로 내보낸다.
2. SVG는 `rsvg-convert`로 PNG 변환한다.
3. PDF는 `pdftoppm`으로 같은 해상도 PNG를 만든다.
4. 페이지별로 rhwp PNG와 PDF PNG를 나란히 배치해 Codex가 직접 시각 판단한다.
5. 결함이 있으면 `dump-pages`, `dump`, 디버그 SVG를 통해 원인을 좁힌다.
6. 공통 로직으로 수정하고 해당 결함을 테스트로 고정한다.
7. `cargo fmt -- --check`, 관련 단위/통합 테스트, 타깃 페이지 PNG 재검증을 통과하면 `task 1274:` 커밋을 만든다.
8. 커밋 후 다음 페이지 또는 다음 문서로 진행한다.

## 완료 기준

6개 대상 문서의 모든 페이지에 대해 PDF 대비 수식 겹침, 미주 간격 차이, page overflow가 없음을 PNG 산출물과 로그로 확인한다. 각 수정 스테이지는 별도 커밋으로 남긴다.
