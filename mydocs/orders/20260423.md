## 8. Firefox AMO 초기 제출

### 버전 bump
- manifest.json / package.json: `0.1.1` → `0.2.1` (rhwp-chrome · rhwp-safari 와 정합)

### 제출 과정 4대 에러 연속 대응
1. **"data_collection_permissions property is required"** — AMO 가 필수화한 키. web-ext lint 경고보다 우선. `required: ["none"]` 복구.
2. **"Unknown strict_min_version 999.0 for Firefox Android"** — placeholder 거부. 공식 방식은 `gecko_android` 키 **완전 생략** (MDN 기준 "desktop only" 자동 선언).
3. **"중복된 부가 기능 ID가 발견됨"** — `rhwp@...` 가 타인 선점. `rhwp-firefox@edwardkim.github.io` 로 변경.
4. **소스 코드 제출 요구** — GitHub URL 로 대체 불가. `git archive` 로 samples 제외 zip (37MB) 제출.
