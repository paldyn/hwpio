### 본 환경 매칭 가드 분석 (`src/renderer/typeset.rs:495-497`)

paragraph 441 매칭:
- exact match: cs/sw 모두 불일치
- any_seg_matches: 모두 불일치
- sw0_match: anchor sw≠0 → 미해당

→ wrap_anchors 미등록 → composer/paragraph_layout 의 LINE_SEG sw 직접 사용 → 좁은 폭 분할.

### Task #722 Stage 8~9 시도 (rollback)
