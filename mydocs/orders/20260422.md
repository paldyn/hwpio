## Task #240 — BMP 임베딩 HWP 문서 SVG 렌더 불가 (버그 수정)

### 배경

`bitmap.hwp`, `한셀OLE.hwp` 샘플에서 SVG 내보내기 시 `data:image/bmp` URI로 BMP 원본이 임베딩되어 브라우저에서 이미지 영역이 비어 보임. SVG `<image>` 요소는 `data:image/bmp` 를 표준 지원하지 않음.

### 수정

- `image` crate 추가(`bmp`, `png` features 최소)
- `bmp_bytes_to_png_bytes` 유틸 추가 (`src/renderer/svg.rs`)
- SVG 임베딩 3개 경로(배경 이미지, `render_image_node`, OLE `native_image`)에 BMP→PNG 재인코딩 적용
- 실패 시 원본 BMP 폴백

### 브랜치 / 상태

- 브랜치: `local/task240` (from `local/devel` ← `devel`)
- 상태: 구현·테스트·보고서 완료, **병합 승인 대기 중**
